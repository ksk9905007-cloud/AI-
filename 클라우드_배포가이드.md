# 🔥 로또 AI - 무료 클라우드(온라인) 배포 가이드

현재 작성된 파이썬 서버 코드(`lotto_server.py`)는 개인 PC 접속, ngrok 터널링뿐만 아니라 **AWS, Render, Heroku 같은 진짜 클라우드 서버 환경**에서도 브라우저(UI) 없이 동작하도록 Headless 기술이 적용되어 있습니다.

가장 쉽고 무료로 파이썬 서버를 24시간 온라인으로 띄우는 방법(Render.com 기준)을 안내합니다.

## 🚀 1단계: 깃허브(GitHub)에 코드 올리기
1. [GitHub](https://github.com/)에 회원가입 후 새로운 저장소(Repository)를 만듭니다.
2. 현재 `모바일로또` 폴더 안에 있는 아래 파일들을 생성한 저장소에 업로드(Commit & Push) 합니다.
   - `lotto_server.py`
   - `lotto_ai.html`
   - `requirements.txt` (필수 설치 패키지 명세서)
   - `build.sh` (클라우드용 크롬 브라우저 자동 설치 스크립트)
   - `render.yaml` (클라우드 자동 세팅 명세서)

## 🚀 2단계: Render.com 클라우드에 연결하기
1. [Render.com](https://render.com/)에 접속하고 GitHub 계정으로 로그인합니다.
2. 대시보드에서 **[New] -> [Web Service]**를 클릭합니다.
3. 방금 코드를 올린 GitHub 저장소(Repository)를 찾아 연동합니다.
4. Render가 코드에 포함된 `render.yaml` 파일을 감지하면 다른 설정 필요 없이 알아서 **무료 파이썬 서버와 내부 크롬 브라우저 구축**을 시작합니다. (최초 빌드 시 5~10분 소요)

## 🚀 3단계: 나만의 온라인 주소(URL) 확인
1. 서버 구축이 완료되면 Render에서 제공하는 고정 클라우드 인터넷 주소(예: `https://lotto-ai-server-xxxx.onrender.com`)가 발급됩니다.
2. 이 주소를 통해 전 세계 어디서든(개인 PC를 꺼두어도) 스마트폰 브라우저로 24시간 로또 시스템에 접속 및 자동 구매 기능을 활용하실 수 있습니다!

> **💡 팁:** 실제 발급된 클라우드 주소 뒤에 접속해보면 여러분의 로또 HTML 화면이 뜨고 정상적으로 서버 연동이 완료되는 것을 확인하실 수 있습니다.
